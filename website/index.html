FAQ

What GPU do I need?

Any GPU with OpenGL 3.3+ support — which covers essentially all hardware made after 2012. Integrated graphics works fine; a discrete GPU gives smoother performance at high resolutions.

Does it work on Apple Silicon (M1/M2/M3/M4)?

Yes. The macOS build is a universal binary that runs natively on Apple Silicon and Intel. Ableton Link, iPhone Continuity Camera, and all 9 engines are fully supported.

macOS says the app is from an unidentified developer — what do I do?

Right-click (or Control-click) the app icon and choose Open on first launch. You'll get a dialog that lets you proceed. After that it opens normally. This is a one-time step required for unsigned apps distributed outside the App Store.

Does it need an internet connection?

No. Everything runs locally — synthesis, audio analysis, MIDI, recording. Ableton Link uses local WiFi/LAN but doesn't need internet access.

Can I use it for live shows and installations?

Yes, absolutely — that's what it's built for. It's free for personal, commercial, and performance use. No attribution required.

Is the source code available?

Yes — it's fully open source on GitHub. Pull requests welcome.
